## Project base module

## Version naming

For project base core module should naming with tag like 

```
{moduleName}/{version}
```

For Example:

| Module Name | Path | Tag Name |
| -- | -- | -- | 
| base | github.com/adon988/project-base/base | base/v0.0.1 |
